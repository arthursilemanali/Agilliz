# 🚚 Agilliz - Logistics ERP 

> **Note:** This repository contains a local copy of the Agilliz project, developed collaboratively by a team of engineering students. This copy is showcased in my portfolio to highlight my specific contributions to the system's architecture, integrations, and frontend structure.

## 📋 About the Project
Agilliz is a comprehensive Enterprise Resource Planning (ERP) platform designed to streamline logistics operations, built with a robust Spring Boot backend and modern frontend technologies. 

## 🤝 My Role & Team Dynamics
I worked as a Fullstack Developer within an agile team. Working in a collaborative environment allowed me to gain practical experience with Git workflows, code reviews, and integrating different modules of a large-scale application.

## 🔑 My Key Contributions
* **System Integrations:** I was responsible for engineering and connecting key API endpoints, ensuring smooth communication between the frontend interfaces and the backend logic.
* **UI/UX & Frontend:** Contributed to the design and implementation of the user interface, focusing on creating a streamlined and intuitive experience for logistics operators.
* **Collaborative Development:** Participated in architectural discussions and code integrations across the `controllers` and `services` layers to maintain a cohesive codebase.

## 🛠️ Overall Project Tech Stack
* **Backend:** Java, Spring Boot, SQL Server, RabbitMQ, JWT Security
* **My Focus Areas:** API Integrations, UI/UX implementation, RESTful bridging.
